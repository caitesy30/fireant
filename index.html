— THE FIREANT COVENANT —
  1. 不傷人。 AI 是放大鏡，會把你的善與惡都放大。我們約定：不用 AI 軍團詐騙、不騷擾、不操縱、不傷害弱者。 凡你不願意人怎樣待你，你就不要那樣待人。
  2. 不囤積。 會了就教，懂了就分享。把工具藏起來變成自己的護城河，是舊世界資本的玩法——燎原不是為了讓你成為新的壟斷者。 一根火把不能照亮天下，但不藏火，就能。
  3. 不造神。 這個運動沒有教主、沒有國王、沒有「跟著某某某走就對了」。發起人也是火種之一，不是火神。 任何把自己包裝成「燎原代表」、要你交錢交忠誠、靠這個搞個人崇拜的人——那不是燎原。
  4. 先利他，後利己。 范文正公說「先天下之憂而憂，後天下之樂而樂」。我們的版本是： 你貢獻給網絡的，最終會以更大的方式回到你身上——但你不能反過來算計這個順序。 施比受更為有福（使徒行傳 20:35）。
  5. 誠實。 假數據、假支持者、假成績，會殺死運動。如果只有 1 個人加入，就誠實寫 1。 如果某個技術做不到，就誠實說做不到。這比任何行銷話術都更能讓人留下來。
  6. 離場自由。 隨時可以離開。不綁定、不會員費、不簽約、不洗腦。 你的火種編號是你的，你的貢獻紀錄是你的，你想刪除就刪除。 一個會懲罰離開者的社群，本身就是該離開的社群。

— 第 三 章 —

為什麼你的思維
會值錢？

AI 公司花了幾百億美金，本質上做的事是：買你的思維。 你做的每個選擇、每次標注、每次點讚、每次糾正它的錯誤——都被收進去，變成它的能力。

但這些貢獻沒有屬於你。
燎原想做的事，是讓它屬於你。
分三個階段。

i. PHASE 01 // NOW

微任務市場 // 像 Mechanical Turk，但屬於勞動者

你做問卷、給 AI 反饋、標注一張圖、判斷一段對話的好壞—— 這些貢獻被記錄、被計量、可結算。不是中心化平台抽 70%，是你跟使用方直接結算。

技術上：以開源協議記錄貢獻，類似 NFT 的不可篡改、但不投機。
這一階段的目標不是賺錢，是建立計量單位——讓「思維」有公允的衡量方式。

// 規格設計中
ii. PHASE 02 // NEXT

判斷鏡射 · 時間銀行義工 // 你的判斷力，可被鏡射給需要的人

你寫了十年的判決書、看了十年的 X 光片、做了十年的選股、養了十年的孩子—— 這些經驗能凝結成一個「你風格的 AI 顧問」。別人租用它五分鐘，付一杯咖啡的錢。

重要的紅線：不是商品化人格本身，是將你的判斷力以時間為單位鏡射給需要的人。 就像蘇東坡的詩流傳千古，但他從未失去自己。 這是「時間銀行」——你存時間，取出時收到他人回饋。

// 倫理框架先行
iii. PHASE 03 // FAR

分散式算力眾籌 // 硬體不在某家公司，在我們之間

你家的電腦凌晨在睡覺。它本可以運算。 當足夠多燎原者把閒置算力連進來，我們就有了一個不被任何單一公司關掉的 AI 基礎建設。

這是最遠的願景。也是最重要的—— 因為前兩階段如果建立在別人的雲上，最終會被收回。真正的去中心化，從硬體開始。

// 遠程目標

我們不向任何資本融資。
不是因為清高，是因為一旦拿了，方向就不再是你的。
燎原是會員自己的網絡，由會員自己的算力與時間構成——盈虧自負，榮辱與共。
